# **App Name**: MortgageMind

## Core Features:

- Mortgage Calculation: Calculate monthly mortgage payment based on loan amount, interest rate, and loan term.
- Scenario Saving: Save mortgage calculation results as a named scenario in local storage.
- Scenario Display: Display saved scenarios in a table format, showing loan details and monthly payment.
- Scenario Comparison: Allow users to compare multiple scenarios side-by-side to see the differences in monthly payments and total interest paid using a tool to predict the best financial fit.
- Scenario Deletion: Enable users to delete saved scenarios from local storage.
- Interest Chart Generation: Generate and display chart comparing how much total interest would be payed for each plan.
- Dynamic Updates: Update calculations dynamically as input values change without needing page reloads.

## Style Guidelines:

- Primary color: Calm blue (#6699CC) to instill trust and security.
- Background color: Light gray (#F0F0F0) for a clean and professional look.
- Accent color: Soft green (#8FBC8F) to highlight key information and actions.
- Body and headline font: 'PT Sans' for clear readability and a modern feel.
- Use a tabbed interface to separate calculation input, scenario display, and comparison views.
- Employ financial icons to represent different data points and actions.
- Provide smooth transitions when switching between tabs or updating calculations.
